       ```bash
           "max_tokens": 100
         }' | python3 -m json.tool
       ```

    5. Create benchmark scripts in `scripts/` directory within the project:

       **scripts/benchmark-latency.py** — Measures Time To First Token (TTFT):
       - Sends a prompt via OpenAI client to localhost:11434
       - Records wall-clock time from request to first token in streaming response
       - Runs 5 iterations, reports mean/min/max TTFT in seconds
       - Uses `time.perf_counter()` for precise timing
       - Model: `qwen2.5-coder:7b-instruct-q4_K_M`
       - Prompt: `"def quicksort(arr):\n    \"\"\"Sort array using quicksort algorithm.\"\"\""`
       - Max tokens: 200
       - Outputs table to stdout: iteration, ttft_seconds
       - Exit code 0 on success, 1 on failure to connect

       **scripts/benchmark-throughput.py** — Measures generation throughput:
       - Sends a prompt via OpenAI client to localhost:11434
       - Uses streaming to measure tokens per second over sustained generation
       - Generates 512 tokens minimum to get stable throughput reading
       - Runs 3 iterations, reports mean/min/max tok/s
       - Model: `qwen2.5-coder:7b-instruct-q4_K_M`
       - Prompt: `"Implement a complete binary search tree in Python with insert, search, delete, and traversal methods. Include docstrings."`
       - Max tokens: 512
       - Calculates tok/s = total_tokens / elapsed_time (excluding TTFT)
       - Outputs table: iteration, total_tokens, elapsed_sec, tokens_per_sec
       - Exit code 0 on success, 1 on failure

    Do NOT create docs/serving-setup.md — that belongs to Plan 02.
