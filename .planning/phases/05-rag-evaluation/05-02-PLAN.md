class PromptAssembler:

    def assemble(self, user_prompt: str, domain: str = "python", max_generation_tokens: int = 2048) -> list[dict]:

    Steps:
    1. Initialize PromptAssembler with model_path
    2. If no_rag is False:
       - Call assembler.retrieve_patterns(domain, limit=5)
       - If verbose: print f"Retrieved {len(patterns)} patterns in {elapsed:.0f}ms"
    3. Call assembler.assemble(user_prompt, domain, max_generation_tokens=max_tokens) -- this already handles RAG injection
       - If no_rag: call assemble after manually setting patterns to empty list (use a flag or pass empty patterns)
       - IMPORTANT: The PromptAssembler.assemble() always calls retrieve_patterns internally. For --no-rag mode, you need to either: (a) add a parameter to assemble() to skip retrieval, or (b) mock the retrieval. Prefer option (a) -- add `skip_rag: bool = False` parameter to PromptAssembler.assemble() that skips the retrieve_patterns call when True. Update prompt_template.py to support this.
    4. Send the assembled messages to Ollama API:
       - Endpoint: http://localhost:11434/v1/chat/completions
       - Model: "qwen2.5-coder:7b-instruct-q4_K_M"
       - Payload: {"model": model, "messages": messages, "max_tokens": max_tokens, "temperature": temperature, "stream": False}
       - Use urllib.request.Request with Content-Type: application/json header
       - Use timeout=120 for the request
    5. Parse response: data["choices"][0]["message"]["content"]
    6. Return the generated content string
    7. On connection error: print error to stderr, return None
    8. On timeout: print timeout message to stderr, return None
