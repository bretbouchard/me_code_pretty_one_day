    **1. pass_at_k function:**
    ```python
    import math

    def pass_at_k(n: int, c: int, k: int) -> float:
        """Calculate pass@k using hypergeometric distribution.
        n: total samples per problem, c: correct samples, k: samples selected.
        """
        if n - c < k:
            return 1.0
        return 1.0 - math.comb(n - c, k) / math.comb(n, k)
    ```

    **2. extract_completion function:**
    ```python
    def extract_completion(prompt: str, raw_output: str) -> str:
        """Extract function body from model output, stripping markdown fences.
        HumanEval expects just the function body that completes the prompt.
        """
        text = raw_output.strip()
        # Strip markdown code fences
        if text.startswith("```"):
            lines = text.split("\n")
            # Remove first line (```python) and last line (```)
            lines = [l for l in lines if not l.strip().startswith("```")]
            text = "\n".join(lines).strip()
        # HumanEval prompt already has the function signature
        # Return the completion (everything after the prompt's last line)
        return text
    ```
